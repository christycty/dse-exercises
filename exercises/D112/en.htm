You are learning bitwise operation. Write a program to help you calculate various types of bitwise operations.

You can learn about bitwise operations from this Wikipedia page.

Input and Output

The only line of input contains two integers $A$ and $B$, separated by a space. ($0 \le A, B \le 100,000$)

On the first line output the bitwise OR of $A$ and $B$.
On the second line output the bitwise AND of $A$ and $B$.
On the third line output the bitwise XOR of $A$ and $B$.


On the fourth line output the bitwise complement of $A$.
